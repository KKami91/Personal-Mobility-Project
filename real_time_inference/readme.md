## streamlit과 안드로이드 앱, python server를 이용한 실시간 탐지 서비스 입니다.


Dependency 설치
```
pip install -r requirements.txt
```

서버 실행
```
python server.py
```

앱 실행
```
streamlit run streamlit-app.py
```

서버와  실행하신 후 app 폴더에 있는 안드로이드 앱을 실행하시면 실시간으로 위법행위에 대한 detect 결과를 보여줍니다.
